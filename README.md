# **Ocean Metaproteomes**
This repository contains metadata and mapping resources for marine metaproteomics samples collected from oceanographic research efforts.

## How to contribute 
Add new datasets
Submit additional marine metaproteomics metadata (with clear provenance and citation information).

Improve metadata quality
Suggest corrections, standardize fields, or expand environmental descriptors (e.g., depth, temperature, nutrients, cruise IDs).
